
# TwitchAlarm
Twitch Alarm for a Huge fan that watching a broadcast is more important than sleeping.

    스트리머: (깜빡이 없이 방송을 키며)방송 받아라!!

    아침
    나: (트위치 알림을 보며) ...응??
